Original Research December 2024

Sleep LLM: A Domain-Specific Language Model for Sleep Science Applications

Chen, J.1, Martinez, S.2, Patel, R.1, Thompson, K.3, Lee, M.1

1Naptick AI, 2Stanford Sleep Medicine, 3MIT Media Lab

Nature Digital Medicine, 2024; 7:425

Abstract

Background: Current approaches to automated sleep analysis rely on general-purpose machine learning models that lack domain-specific knowledge. We present Sleep LLM, the first large language model specifically trained for sleep science applications.

Methods: We fine-tuned a transformer-based architecture on a curated dataset of 120+ million nights of multimodal sleep data, including polysomnography, wearable sensors, sleep diaries, and clinical notes. The model was trained using a novel temporal attention mechanism optimized for biosignal processing.

Results: Sleep LLM achieved 94.2% accuracy in sleep stage classification (Cohen's Œ∫ = 0.91), 96.8% sensitivity in apnea detection, and 98.1% agreement with board-certified sleep specialists in diagnostic assessments. The model demonstrated superior performance across all metrics compared to existing approaches.

Conclusions: Domain-specific language models represent a significant advancement in automated sleep analysis, offering accuracy comparable to human experts while enabling new applications in personalized sleep health.

Evaluation Protocol

  • All models evaluated on held-out test sets with no patient overlap
  • Human expert scores averaged across 3 board-certified sleep specialists
  • Statistical significance tested using paired t-tests (p < 0.001 for all comparisons)
  • Code for reproducing benchmarks available on GitHub

Research Methodology

Data Collection & Curation

Our training data comes from diverse sources including academic sleep centers, consumer wearables, and clinical trials. All data is de-identified and processed following HIPAA guidelines.

  • Multi-site data collection from 15 sleep centers
  • Automated quality control with manual verification
  • Balanced representation across demographics
  • Continuous data augmentation pipeline

Model Architecture

Sleep LLM uses a custom transformer architecture optimized for temporal biosignals with innovations in attention mechanisms and multi-scale processing.

  • Hierarchical temporal attention layers
  • Multi-resolution signal encoders
  • Cross-modal fusion modules
  • Efficient inference optimization

Validation Framework

Rigorous validation through multiple clinical studies ensures our model meets the highest standards for medical AI applications.

  • Multi-center prospective validation
  • Comparison with gold standard PSG
  • Inter-rater reliability assessment
  • Continuous monitoring post-deployment
